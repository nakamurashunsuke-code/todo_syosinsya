# todo_syosinsya

Supabase連携のTODOアプリです。  
静的サイトとしてGitHub Pagesに公開できます。

## 1. Supabaseテーブル作成

SupabaseのSQL Editorで以下を実行してください。

```sql
create table if not exists public.todos (
  id bigint generated by default as identity primary key,
  text text not null,
  completed boolean not null default false,
  created_at timestamptz not null default now()
);
```

## 2. config.js を設定

`config.example.js` を参考に、`config.js` の値を自分のプロジェクト情報に変更します。

```js
window.APP_CONFIG = {
  SUPABASE_URL: "https://YOUR_PROJECT_REF.supabase.co",
  SUPABASE_ANON_KEY: "YOUR_SUPABASE_ANON_KEY",
};
```

## 3. 動作確認

`index.html` をブラウザで開いて、TODOの追加・完了・削除ができればOKです。

## 4. GitHub Pagesで公開

このリポジトリでは `main` ブランチの `/` をPages公開に設定済みです。  
`git push` 後に数分待つと、公開URLで最新版が反映されます。
